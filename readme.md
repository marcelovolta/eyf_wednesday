# Proyecto Wednesday

Este proyecto tiene como objetivo plasmar en código todo lo visto durante las clases de los miércoles en el segundo cuatrimestre de 2025 en la clase de EyF. 
El objetivo del proyecto es construir un template escrito en Python que sirva como base para cualquier proyecto de investigación, aplicando buenas prácticas de aceptación en la industria. 
Dentro del directorio docs voy a insertar documentos md con notas extraídas de las clases sobre los puntos que  me parezcan más importantes. 
El formato de proyecto es scripts de Python que pueden correrse sin un notebook de por medio. 
Por otra parte, se prioriza el uso de DuckDB con Pandas para la manipulación de los datos. La idea es que el código es más claro e intuitivo de este modo. Pandas adolece de una complejidad creciente a medida que las expresiones usadas tratan de conseguir efectos de transformación con más requerimientos. 
Por otra parte, las funciones se nombran en Inglés, dado que la idea es poder compartir el código de futuros proyectos con la comunidad en general y el Inglés es el idioma de preferencia.
